Как и на чем работает этот блог

  1. Это полностью статический сайт, опубликованный с помощью Hugo. Статика позволяет снизить нагрузку с сервера и хранить сие на любом самом дешёвом хостинге.
  2. На сайте включен https сертификатом от Let’s Encrypt. Инструкция по настройке для Azure Web App.
  3. Все это находится в Azure. Служба приложения S1, это на 99% больше чем нужно для данного сайта, но со временем будут работать внешние воркеры в данной службе.
